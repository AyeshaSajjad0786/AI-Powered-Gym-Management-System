# AI-Powered Gym Management System

**OOAD Semester Project** | *Unified Process · SOA · UML*

##  Overview

Full-scale, AI-integrated Gym Management System designed using **Unified Process** across all 4 phases. Architected with **Service-Oriented Architecture (SOA)** and 4 distinct layers.

## Key Features
- AI-powered personalized workout & diet recommendations
- Automated billing & scheduling with conflict detection
- Real-time progress dashboards
- 24/7 AI chatbot support
- GDPR/HIPAA compliance, 2FA, role-based access

## 8 Modules (Increments)
1. Authentication & Member Management
2. Admin Dashboard
3. Trainer Management
4. Scheduling & Booking
5. Workout Planning
6. Nutrition & Diet
7. Billing & Payments
8. Communication & AI Chatbot
